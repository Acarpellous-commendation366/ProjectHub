## 🌐 What Is ProjectHub?

ProjectHub is a web app that helps you organize projects and tasks. It works like a simple version of tools such as Trello or Notion. You can create projects, add tasks, set deadlines, and track progress in one place. The interface is made for anyone, even if you don’t have a technical background.

You can use ProjectHub to:

- Keep track of your work and personal projects
- Manage tasks with clear lists and cards
- Organize projects by deadline or priority
- Collaborate easily if you want to share your progress

## ❓ Troubleshooting
